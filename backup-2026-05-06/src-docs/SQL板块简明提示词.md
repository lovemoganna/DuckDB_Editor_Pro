# SQL 板块简明优化提示词

请在现有系统基础上，基于 MECE 原则进一步优化 SQL 编辑器模块。

**目标**：实现 SQL 编写、执行与调试能力的快速调用，并根据用户输入即时生成对应的 DuckDB SQL 方案。整体结构需清晰、调用路径明确，确保在复杂场景下稳定输出可执行的 SQL 逻辑。

**模块优化要求（适用于各子模块）**：

1. 内嵌「AI 一键填充」功能，基于当前选中的表/列上下文自动生成 SQL 查询草稿，支持智能填充、片段模板（CTE / 窗口函数 / PIVOT 等）与自然语言三种填充模式，降低用户冷启动门槛。
2. 内嵌「快速清除」按钮，支持一键重置 SQL 编辑器内容与 AI 输入框，并保留 Ctrl+Z 撤销能力，提升多轮迭代效率。
3. 提供模块背景说明，按 MECE 五类结构分组（SQL 生成 / 数据分析 / 数据转换 / 性能调优 / 实用工具），每类包含使用场景、常见错误与 AI 协作提示，支持用户基于提示与 AI 进行二次优化。
